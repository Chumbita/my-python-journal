Proyecto 21:
Este programa es un promediador de notas de un curso académico. Ese programa cuenta con tres opciones que el usuario podrá realizar:
    1. Cargar nombre del estudiante y sus respectivas notas.
    2. Mostrar listado de estudiantes con sus respectivas notas.
    3. Mostrar listado notas finales en donde se muestra el estudiante y el promedio de sus notas. 
En la opción (1) el usuario será capaz de ingresar la cantidad de notas evaluativas con las que cuenta su curso. La carga de estudiantes y sus notas se realizan uno por uno. Después de cada carga se le pregunta si desea ingresar otro estudiante.

Proyecto 21:
This program is a grade averaging tool for an academic course. It features three options that the user can perform:
    1.Enter the student's name and their respective grades.
    2.Display a list of students with their respective grades.
    3.Display a final grades list showing each student and the average of their grades.
In option (1), the user will be able to enter the number of evaluative grades in their course. The entry of students and their grades is done one by one. After each entry, the user is asked if they want to enter another student.
