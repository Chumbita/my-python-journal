Proyecto 30: 

REQUERIMIENTOS:
-Crea una clase Producto con atributos como nombre, precio, y categoria.
Implementa una clase Inventario que maneje una colección de productos.
Funcionalidad de Filtrado:

-En la clase Inventario, implementa un método que filtre los productos según su categoría y su precio.
Permite que el usuario elija una categoría y un rango de precios, y luego devuelve solo los productos que cumplan esos criterios.

-Implementa un método que aplique un descuento a todos los productos de una determinada categoría y que estén dentro de un rango de precio.
El descuento se debe aplicar utilizando map y debe retornar una lista con los precios actualizados.

-Diseña una interfaz de línea de comandos que permita al usuario interactuar con el inventario: listar productos, aplicar filtros, y aplicar descuentos.


Project 30: 

REQUERIMENTS:
-Create a Product class with attributes such as name, price, and category. Implement an Inventory class that manages a collection of products.
-In the Inventory class, implement a method that filters products by category and price. Allow the user to select a category and a price range, and then return only the products that meet those criteria.

-Implement a method that applies a discount to all products in a certain category and within a specific price range. The discount should be applied using map and should return a list with the updated prices.

-Design a command-line interface that allows the user to interact with the inventory: list products, apply filters, and apply discounts.
