Proyecto 5:
Se crea una función que toma una lista y un número entero como parámetro y devuelve la lista rotada hacia la derecha 'n' lugares. Por ejemplo, si la lista es [1, 2, 3, 4, 5] y n es 2, la lista resultante debería ser [4, 5, 1, 2, 3].

Proyect 5: 
A function is created that takes a list and an integer as parameters and returns the list rotated to the right 'n' places. For example, if the list is [1, 2, 3, 4, 5] and n is 2, the resulting list should be [4, 5, 1, 2, 3].
