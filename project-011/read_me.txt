Proyecto 11:
Se creó un programa que almacena en un diccionario los precios de unas frutas. El usuario pregunta por una fruta, ingresa el número de kilos y se muestra por pantalla el precio de ese número de kilos de fruta. Si la fruta no se encuentra en el diccionario se notifica al usuario.

Project 11: 
A program was created that stores the prices of fruit in a dictionary. The user asks for a fruit, enters the number of kilos and displays the price of that number of kilos of fruit. If the fruit is not in the dictionary the user is notified.
