Proyecto 4: 
Se crea una función que toma dos listas como parámetro y devuelve una lista que contenga elementos alternados de ambas listas. Por ejemplo si ambas listas tienen el mismo tamaño [1, 2, 3] y [4, 5, 6], la lista resultante debería ser [1, 4, 2, 5, 3, 6], pero si ambas listas no tienen el mismo tamaño [1, 2, 3] y [4, 5, 6, 7, 8, 9], la lista resultante debería ser [1, 4, 2, 5, 3, 6, 7, 8, 9].

Project 4: 
A function is created that takes two lists as parameters and returns a list containing alternating elements from both lists. For example, if both lists have the same size [1, 2, 3] and [4, 5, 6], the resulting list should be [1, 4, 2, 5, 3, 6]. However, if both lists do not have the same size [1, 2, 3] and [4, 5, 6, 7, 8, 9], the resulting list should be [1, 4, 2, 5, 3, 6, 7, 8, 9].
