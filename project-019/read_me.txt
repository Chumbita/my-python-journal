Proyecto 19:
  Se implementó una función principal que actúa como el menú de una calculadora científica. A través de esta función, el usuario puede elegir realizar varias operaciones matemáticas, como seno, coseno, tangente, exponencial y logaritmo neperiano. Para facilitar estas operaciones, se importó la biblioteca math.

  Para cada una de las operaciones matemáticas, se creó una función dedicada. Estas funciones generan e imprimen en pantalla una tabla que muestra los enteros del 1 hasta el valor introducido por el usuario, junto con el resultado de aplicar la función matemática correspondiente a cada entero.

Project 19: 
  A main function was implemented that acts as the menu for a scientific calculator. Through this function, the user can choose to perform various mathematical operations, such as sine, cosine, tangent, exponential, and natural logarithm. To facilitate these operations, the `math` library was imported.

  For each mathematical operation, a dedicated function was created. These functions generate and print a table displaying the integers from 1 to the value entered by the user, along with the result of applying the corresponding mathematical function to each integer.
